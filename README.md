# Velio
Velio is a eCommerce Blazor Application.

# Planned Features
- Basket.
- Product ratings and reviews.
- Product categories.
- Product tags. (i.e "Free-Shipping", "On-Sale")
- Search.
- Independent login page for users/admin.
- Token-based authentication.
- Admin CRUD for products.
- Mudblazor UI.
